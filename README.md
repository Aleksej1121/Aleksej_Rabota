# Aleksej_Rabota
 
Репозиторий, содержащий приложение для учета мероприятий и используемого оборудования, созданное на Flask

**Приложение позволяет:**
- Просматривать полную информацию о запланированных мероприятиях и доступном оборудовании
- Добавлять, редактировать и удалять записи через две удобные веб-формы
- Закреплять конкретное оборудование за определенными мероприятиями
- Контролировать наличие и статус техники для каждого события

**Особенности:**
- База данных создана на SQLite
- Реализованы 2 ключевые формы: форма управления мероприятиями и форма учета оборудования
- Автоматическое заполнение базы данных первичными тестовыми данными при первом запуске
- Быстрая фильтрация мероприятий по дате проведения и типу используемого оборудования
- Четкое разделение контроллеров, моделей и представлений (архитектура MVC)
- Полный набор тестов на pytest для проверки работы форм и логики базы данных. Тесты проходят в изолированной БД SQLite in-memory и не затрагивают основные данные

**Структура проекта:**
controllers/ (Обработка запросов, валидация форм и связь с моделями)
models/ (Схемы таблиц мероприятий и оборудования, логика БД)
views/ (Шаблоны страниц и интерфейс форм)
static/ (Стили, скрипты и элементы дизайна)
/css/ (Стилизация интерфейса и форм)
/js/ (Интерактивная логика для работы с формами)

**Установка**
1. Перейдите в любой удобный каталог, клонируйте репозиторий и перейдите в папку проекта: и перейдите к нему и установите библиотеки
- `pip install flask pytest flask-sqlalchemy pymysql cryptography flask-login werkzeug`

2. Запустите приложение
- `run.py`
- При первом запуске программы app.py автоматически создаст базу данных и внесет пользователя с именем nim и паролем lol

3. Перейдите по ссылке (http://127.0.0.1:5000)

**Тестирование**
1. Перейдите в корень проекта
2. Запустите тесты
- `pytest -v`
Тесты используют SQLite.
